# Running the library

<Steps>

### Use Node 16

Make sure your node version is up-to-date. It should be at least **Node 16.14.0:**

```sh
  ## Check your node version:
  node -v  ## Output: v14.18.3


  ## Use Node 16
  nvm use v16
```

### Install pnpm

If you don't already have pnpm installed, you can install it with:

```bash
npm install -g pnpm
```

### Install dependencies

This command installs all dependencies for all projects (Example folder, docs folder, root folder & components library folder)

```sh
pnpm i
```

### Running the Next.js playground:

```sh
pnpm run:playground
```

### Running the docs:

```sh
pnpm run:docs
```

</Steps>

### Additional commands

Delete lib node_modules & build:

```bash
pnpm clean
```

Check for Typescript type errors on the whole project:

```bash
pnpm typecheck
```
